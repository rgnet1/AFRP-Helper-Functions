What is Preprocessing?

Preprocessing templates allow you to transform and clean your data automatically. Use Value Mappings for exact text replacements (e.g., "Steak" → "S") and Contains Mappings to remove text that appears anywhere in a field (e.g., "- Reserved" → "").

Data Transformations

Value Mappings (Exact Match)

Replace exact text values. For example, shorten meal preferences or simplify event names. Both fields support free text.

Examples:
  • "Steak" → "S" (Shorten meal preference)
  • "No Club Affiliation" → " " (Remove text)
  • "Mid-Year Meeting 2026" → "Lexington 2026" (Simplify name)

Contains Mappings (Substring Match)

Remove text that appears anywhere in a field. Useful for cleaning up notes, prefixes, or suffixes. Both fields support free text.

Examples:
  • "- Reserved" → "" (Remove reservation note)
  • "Ramallah Federation in " → "" (Remove prefix)
  • "(Ages 13-17)" → "" (Remove age restriction)
